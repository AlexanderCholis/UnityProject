Setup Instructions:

1. Set rendering path to linear under Player Settings
2. Download and install Unity Post Processing Stack on the Unity Asset Store: https://assetstore.unity.com/packages/essentials/post-processing-stack-83912
3. Add the Post Processing Stack to the main camera and drag the provided profile named "SeaViewPProfile_01" into the open slot
4. (*Optional) Download and install the freeHDRI asset available for free on the Unity Asset Store and drag the preferred HDR into your "Skybox Material" located under Lighting settings


General Tips:

- Make use of the prefabs for a easier workflow
- Under Custom Prefabs-> Custom you will a selection of premade assets to iterate concepts fasters
- Disable shadows on the point lights located under the "LED_PointLights" gameobject to improve performance.
